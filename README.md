# E-Commerce Capstone - React

**Description**
A simple e-commerce demo built with React, using FakeStore API. Features: product listing, search/sort/filter, product details, cart (add/remove/update), checkout with validation, light/dark theme, responsive layout.

**Tech**
- React + Vite
- React Router
- Zustand (global store)
- Axios
- React Hook Form + Yup
- Tailwind CSS (optional)

**Run locally**
```bash
npm install
npm run dev
